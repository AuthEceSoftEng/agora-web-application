AGORA is a Code Search Engine that offers the most advanced features in order to help developers find the source code that they need.

AGORA offers three types of search:
  • Simple Search
    Searches the whole directory for the search terms provided.
  • Advanced Search
    Provides syntax-aware search for source code. One can search using class names or modifiers (e.g. public), variables, method names and return types, parameters (and their respective modifiers), etc.
  • Snippet Search
    Allows searching by giving code snippets as search terms.
  • Project Search
    Allows searching for projects given class names, and other elements (implements and extends) so that small patterns can be found.

Additionally AGORA offers an open RESTful API for searching.
