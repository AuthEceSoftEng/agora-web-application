AGORA is a search engine for source code that offers the most advanced features in order to help developers find the source code that they need. Apart from simple code search, AGORA provides a number of different options including advanced syntax-aware search, snippet search and search for projects. Additionally an open API is provided for any developers wishing to use the service by creating their own client.

The term "agora" refers to the central spot of ancient Greek city-states. Back then, "agora" was more than a "market", it was a place where people met to trade goods but also exchange ideas. It is a good fit for our search engine since we envision it as a place where developers can (freely) distribute and exchange their source code, and subsequently their ideas.

AGORA has been developed in the Intelligent Systems & Software Engineering Lab (ISSEL) of the School of Electrical & Computer Engineering of the Aristotle University of Thessaloniki.
